# Personal-Python-Notes
## Getting help
* `help`. For instance, `help(len)`, or `help(tuple.index)` for known attributes.
* `dir`. For instance, `dir(list)` for all attributes for lists.

## Basic functions
* `type` for getting the data type of the object in Python.
